# Next Frontend Bootstrap

This folder starts the migration to a Next.js frontend with reusable views/components.

## Run

1. Install deps:
   - `cd next-frontend`
   - `npm install`
2. Start backend API in project root:
   - `npm run dev`
3. Start Next frontend:
   - `cd next-frontend`
   - `npm run dev`

Next runs on `http://localhost:3001`.
API calls to `/api/*` are proxied to `http://localhost:3000/api/*`.
